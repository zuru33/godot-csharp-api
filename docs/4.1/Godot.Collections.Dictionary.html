    Class Dictionary

    Duplicate(bool)

    Returns a copy of the Dictionary. If deep is true, a deep copy is performed: all nested arrays and dictionaries are duplicated and will not be shared with the original dictionary. If false, a shallow copy is made and references to the original nested arrays and dictionaries are kept, so that modifying a sub-array or dictionary in the copy will also impact those referenced in the source dictionary. Note that any GodotObject derived elements will be shallow copied regardless of the deep setting.
